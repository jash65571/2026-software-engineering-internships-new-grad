# Contributing

Thanks for helping keep this job list useful.

## Submit a job

Open an issue or a pull request with:

- Company
- Role title
- Location
- Internship or new grad
- Apply link (must go to the company's own careers page when possible)
- Date added
- Notes (sponsorship, remote, deadline)

## Rules

Please do not submit:

- Expired jobs
- Fake jobs or scams
- Jobs with broken apply links
- Commission-only roles
- Vague recruiter posts with no real description
- Duplicate jobs already in the list

## Quality bar

A submission should be useful to a student or new grad applying to software engineering roles. If the post looks suspicious, open a separate issue tagged `suspect` instead of adding it to the main list.

## Style

- Keep rows short. One row per role.
- Date format: `YYYY-MM-DD`.
- Use a real apply link, not a tracking link or aggregator URL.
- One pull request per batch is fine; please don't open dozens of one-line PRs.
