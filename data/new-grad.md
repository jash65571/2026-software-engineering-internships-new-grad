## New grad resume tips

For new grad roles, your resume should show:

- Projects with measurable impact
- Internships
- Technical skills that match the job description
- GitHub or portfolio link
- Clear, specific bullet points
- Keywords from the job description

## Common new grad role titles

- Software Engineer, New Grad
- Software Development Engineer I
- Associate Software Engineer
- Junior Software Engineer
- Software Engineer, University Graduate
- Associate Member of Technical Staff (Salesforce)
- Member of Technical Staff (some startups)
